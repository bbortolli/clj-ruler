# Getting Started

## Installation

Leiningen
```clj
[org.clojars.bbortolli/ruler "0.1.0-SNAPSHOT"]
```

Deps EDN
```
org.clojars.bbortolli/ruler {:mvn/version "0.1.0-SNAPSHOT"}
```

Maven
```xml
<dependency>
  <groupId>org.clojars.bbortolli</groupId>
  <artifactId>ruler</artifactId>
  <version>0.1.0-SNAPSHOT</version>
</dependency>
```
